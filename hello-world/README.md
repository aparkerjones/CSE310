# Overview

I created this project to keep building confidence in C# and to get better at following a clean software workflow from start to finish. I want to be consistent with how I organize, compile, and share my code so I can carry that into bigger projects.

This software is a simple C# console application that prints `Hello World` to the terminal.

The main purpose of this software is to confirm that my development environment is working correctly and to give me a solid baseline project I can build on in future modules.

# Development Environment

- Visual Studio Code
- .NET SDK 10.0
- PowerShell terminal

Programming language used:
- C#
